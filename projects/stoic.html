BrickMan Superhero Project

Context

The Movie Webpage, primarily called BrickMan, is created with the help of me (Rain) and Jehan (my partner). We used our knowledge of HTML and CSS to create an tradegy-based webpage movie.

Content

There's no good project without a plan so me and my partner created a text file, plan.md, md standing for markdown. There are requirements for the movie webpage like elements and selectors in the webpage we have to agree on, and specialize in it. Beginning with HTML requirements:

  • headers (Jehan)
  • paragraphs (both)
  • images (Rain)
  • imported and Original Google Fonts (Rain)
  • divs and spans (Rain)
  • inputs and submit (Jehan)

CSS requirements:

  • all selectors (Jehan)
  • Box Model- padding, margin, border properties (Rain)
  • images (Rain)
  • Background and color scheme (Jehan)
